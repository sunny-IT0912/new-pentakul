## ✨ Features

### Core Features
- ✅ **Hero Section** with looping video background
- ✅ **Sticky Card Scrolling Effect** - Cards stack and cover each other as you scroll
- ✅ **Dancing/Bouncing Card Animations** - Cards fly in with continuous bounce effect
- ✅ **Animated Counters** - Numbers count up when scrolled into view
- ✅ **Split Layout Sections** - Sticky left content with scrollable right content
- ✅ **Accordion FAQ Dropdowns** - Expandable questions with smooth animations
- ✅ **Parallax Scrolling Image** - Image reveals different parts based on scroll direction
- ✅ **Horizontal Scrolling Bands** - Two bands moving in opposite directions on scroll
- ✅ **Search Functionality** - Filter industries by name
- ✅ **Responsive Navigation** - Desktop dropdown menus and mobile hamburger menu
- ✅ **Sticky Header** - Hides on scroll down, appears on scroll up
- ✅ **Sticky Vertical Logo** - Brand logo fixed on left side throughout the website

### Pages

| Page | Description | Key Features |
|------|-------------|--------------|
| **Home** | Main landing page | Video hero, sticky service cards, achievements counter, testimonials, certifications carousel, awards, global footprint |
| **Industries** | Industries we serve | Search bar, dancing card grid, FAQ accordion, parallax image, tailored logistics CTA |
| **Pentakuhl** | Temperature-controlled solutions | Video hero, product solutions (sticky effect), series guide, horizontal bands, FAQ |

## 🛠️ Tech Stack

### Frontend
- **React.js** (v18.2.0) - UI library
- **React Router DOM** (v6.14.0) - Navigation & routing
- **Framer Motion** (v10.12.0) - Animations & transitions
- **Tailwind CSS** (v3.3.0) - Styling & responsive design

### Build Tools
- **Vite** (v4.3.0) - Fast build tool and development server
